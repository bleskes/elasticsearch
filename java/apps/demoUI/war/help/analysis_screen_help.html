Yellow Dot Explanation

Now we see that the previous dip in Idle Thread Count corresponds to the Yellow dot we observed at the onset of this demo.

Note that many of the same metrics involved in the Red Dot activity are detailed here. This Yellow Dot is a 'wobble' or pre-cursor behavior to the Red Dot that likely did not raise alarms.

Prelert Real-Time's predictive capabilities identify these pre-cursor activities that often occur hours before a major incident providing plenty of time to resolve it before users are impacted.
